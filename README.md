## Tecnologías utilizadas
- React.js
- JavaScript
- CSS
- HTML

## Instrucciones para obtener la app

- Clonas el repo
- Entras a la carpeta
- Instalas dependencias si es necesario

## Utilidades dentro de la app

- Selecciona un día (Lunes a Domingo).
- Ingresa el nombre de la tarea (obligatorio).
- Selecciona la fecha (obligatorio).
- Agrega notas si deseas (opcional).
- Presiona Agregar para guardar.
- Presiona Eliminar para borrar una tarea.
